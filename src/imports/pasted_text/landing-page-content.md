FLUXOS IMPORTANTES DENTRO DO PRODUTO
Fluxo A — Novo cliente

Landing → Solicita acesso → conta criada → login → onboarding → painel admin

Fluxo B — Admin monta a loja

Login → onboarding → personalização → produto → conteúdo → compartilhar

Fluxo C — Admin vende/divulga

Painel → gerar conteúdo → copiar → divulgar → atrair clique → loja pública

Fluxo D — Super Admin gerencia tudo

Login → dashboard super admin → admins/lojas/planos/métricas
